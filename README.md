# DICOM Segmentation Tool

Приложение с графическим интерфейсом для просмотра медицинских DICOM-изображений с наложенными сегментациями в формате JSON.

## Возможности

- **Загрузка DICOM файлов** — автоматическое определение параметров пикселя (PixelSpacing) для точных измерений
- **Отображение сегментаций** — визуализация разметки из JSON-файлов (формат совместимый с LabelMe)
- **Поддерживаемые фигуры:**
  - Полигоны (с автоматическим расчётом площади)
  - Прямоугольники
  - Круги и эллипсы
  - Линии и точки
- **Линейка** — интерактивный инструмент для измерения расстояний на изображении
- **Автоматический расчёт площади** — в мм² для всех замкнутых фигур

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
python measure_gui.py
```

1. Нажмите "Выбрать папку" и укажите директорию с парами файлов DICOM + JSON, например `P0001.dcm` + `P0001.json`
2. Выберите пару из списка
3. Используйте кнопку "Линейка" для измерения расстояний
4. Кликайте на сегментации в таблице для их выделения

## Структура данных

```
папка/
├── P0001.dcm      # DICOM-изображение
├── P0001.json     # Файл разметки
├── P0004.dcm
├── P0004.json
└── ...
```

## Сборка исполняемого файла

```bash
pyinstaller measure_gui.spec
```

## Зависимости

- Python 3.9+
- pydicom — работа с DICOM файлами
- Pillow — обработка изображений
- numpy — вычисления
